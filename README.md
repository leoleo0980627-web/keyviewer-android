# KeyViewer for Android

**vibe coding**

KeyViewer 是一款基于 Shizuku 的 Android 键盘按键可视化工具。能够悬浮显示自定义的键盘布局，实时统计按键次数，支持键雨特效、按键映射、图层管理、复制粘贴等丰富功能。

## 预览截图

![主界面预览](https://github.com/leoleo0980627-web/keyviewer-android/blob/main/Screenshot_20260423-012854_KeyViewer.png)

## 主要功能

### 基础功能

- **悬浮窗显示**：将自定义的按键布局以悬浮窗形式展示，适用于直播、录屏等场景。支持缩放调整悬浮窗大小。
- **自定义按键布局**：自由添加、删除、拖拽调整按键位置，支持网格吸附对齐。
- **按键绑定**：通过 Shizuku 监听物理键盘事件，将物理按键与可视化按钮绑定。在设置面板中点击绑定按钮后按下物理按键即可绑定。
- **按键映射**：将物理按键映射为其他按键输出。需在系统设置中启用 KeyViewer 输入法，启用后点击左下角「切换输入法」按钮弹出系统输入法切换列表。映射按键通过内置输入法发送，不影响正常打字。
- **实时计数**：统计每个按键的按下次数，支持显示 KPS（每秒按键数）和 TOTAL（总按键数）。
- **手势操作**：支持双指缩放、长按拖动画布、单点拖拽移动按键、拖拽边角调整按键大小。

### 特殊格子（动态显示）

在自定义显示文字中输入特定关键词，格子会显示动态内容：

- `KPS`：显示当前每秒按键数，格式为「KPS / 数字」，实时更新。
- `TOTAL` 或包含 `tot` 的文字：显示总按键次数，格式为「文字 / 数字」，实时更新。

### 外观定制

- **尺寸与样式**：自由调整按键宽度、高度、圆角、边框宽度。
- **颜色设置**：自定义按键填充色、边框色、文字颜色。
- **文字设置**：自定义显示文字、文字大小，可选择是否显示按键次数。
- **文字颜色**：独立设置文字颜色，支持透明度（ARGB）。
- **字数自适应**：当显示数字位数较多时（4位以上），仅数字行自动缩小字号，键名保持原大小。

### 键雨特效

- 每个按键可独立开启键雨效果。
- 支持调整键雨的 X/Y 偏移位置、宽度百分比、颜色、上升速度。
- 按键按下时键雨从按键底部持续增长，松开后粒子下落消失。
- 键雨粒子遵循图层顺序，高图层按键的粒子在低图层之上。

### 图层管理

- **图层管理面板**：右上角图层图标打开，显示所有按键列表。
- **拖拽排序**：长按列表项右侧三条横线图标拖动，调整按键图层顺序，实时生效。
- **点击选中**：点击列表项直接选中对应按键并打开设置面板。
- **自动排列**：按位置自动排序，坐标靠下靠左的按键在上层。
- **重置图层**：一键清除所有自定义图层设置，恢复为 0。

### 全局设置

- **悬浮窗缩放**：调整悬浮窗显示比例（50%~200%）。
- **键雨最大高度**：设置键雨粒子最大下落高度。
- **重置所有按键计数**：将所有按键计数归零，KPS 和 TOTAL 不受影响。
- **重置悬浮窗位置**：将悬浮窗位置还原到默认。
- **校准键盘设备**：按下物理按键自动识别键盘设备路径，用于解决多键盘设备监听问题。
- **关闭按钮**：全局设置面板有关闭按钮，点击带动画效果等同于按返回键。

### 配置管理

- **导出配置**：将按键布局、外观设置、计数数据导出为 JSON 文件。当前版本文件保存可能为空，建议使用长按导出按钮分享配置文本。
- **导入配置**：一键恢复之前备份的配置，导入后按键自动对齐网格，计数数据直接替换无需等待。
- **长按分享**：长按导出按钮可直接分享配置 JSON 文本到其他应用。

### 多选与复制粘贴

- **多选模式**：点击工具栏多选按钮进入，可点击多个按键进行选择。
- **批量删除**：多选后点击删除按钮批量删除选中的按键。
- **复制**：多选模式下选中至少一个按键后，工具栏出现复制按钮，点击复制选中按键的所有属性。
- **粘贴**：复制后粘贴按钮变为可用，点击进入粘贴模式。粘贴的按键以 40% 透明度出现在画面中央，可整体拖动调整位置，松手自动对齐网格。工具栏切换为确认和取消按钮，确认后正式放置，取消则丢弃。

### 其他

- 左下角「开启悬浮窗」按钮，开启后悬浮窗显示在所有应用之上。
- 左下角「授权 Shizuku」按钮，点击授权或显示当前 Shizuku 连接状态。
- 左下角「切换输入法」按钮，点击弹出系统输入法切换列表。若未启用 KeyViewer 输入法则弹窗引导去系统设置启用。
- 首次启动或开启悬浮窗时若输入法未启用，弹窗引导用户启用。
- 关闭悬浮窗时自动切换回原输入法。
- 多个输入设备的事件通过 FIFO 管道汇聚监听，避免多进程性能问题，关闭时自动清理。

## 使用说明

1. 安装 Shizuku 并授予权限。
2. 打开 KeyViewer，点击「授权 Shizuku」按钮完成授权。
3. 如需使用按键映射功能，需在系统设置中启用 KeyViewer 输入法（首次开启悬浮窗时会弹窗引导）。
4. 使用底部工具栏添加按键，点击按键打开设置面板进行绑定和外观调整。
5. 点击右上角齿轮图标打开全局设置，可调整悬浮窗缩放、导出配置、重置计数、校准键盘等。
6. 点击右上角图层图标打开图层管理面板，可拖拽调整按键层级、自动排列。
7. 点击「开启悬浮窗」将布局以悬浮窗形式显示（需授予悬浮窗权限）。

## 已知问题

- 导出配置保存到文件可能为空文件，临时解决方案：长按导出按钮直接分享配置文本。
- 按键映射功能基础可用，部分组合键暂不支持。
